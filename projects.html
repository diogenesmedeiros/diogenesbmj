Olha o Trabalho Aí

"Olha o Trabalho Aí" é uma plataforma que conecta prestadores de serviços com clientes na mesma região, facilitando a comunicação e o pagamento, incluindo a aceitação de PIX na própria plataforma. O projeto utiliza tecnologias modernas como Node.js, Express, e Sequelize para o backend, com integração de WebSockets via Socket.IO para comunicação em tempo real. No frontend, utilizamos React para uma interface interativa e dinâmica.
